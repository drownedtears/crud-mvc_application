-Simple crud application with mvc pattern-
Java version - 17
You can see all users, create new user, delete or update any user.
After starting you can follow instructions or use this:
1. http://localhost:8080/crudApplication/ - welcome page
2. http://localhost:8080/crudApplication/users - all users list
3. http://localhost:8080/crudApplication/users/add - addition of new user
4. http://localhost:8080/crudApplication/users/1 - find user by id=1
